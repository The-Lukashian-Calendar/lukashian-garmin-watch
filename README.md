This is a Watch Face for Garmin smartwatches. It displays the current time, day and year of the Earth instance of the Lukashian Calendar.

To download it onto your Garmin smartwatch, please visit the [Garmin App page](https://apps.garmin.com/apps/60848d83-5c25-4a6b-a5de-2f5cb6d45bdb).

For more information on the calendar mechanism itself, please visit [lukashian.org](https://www.lukashian.org)!
